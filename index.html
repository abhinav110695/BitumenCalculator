Bitumen Calculator

Safari-friendly single page web app with interpolation for VG-10 Dip at Centre Pipe and 30 MT Cylindrical Tanker quantity lookup.

1. VG-10 Dip at Centre Pipe

Enter only inches, only feet, or both. Quantity is calculated using exact match or linear interpolation between the nearest chart points.

2. Cylindrical Tanker (30 MT)

Calculate quantity using Dip or Empty Space. Values between chart points are interpolated automatically.

Input Logic Used

Only inches entered

Feet is treated as 0. You can enter 9, 15, 27, or decimal inches like 27.5.

Only feet entered

Inches is treated as 0. Decimal feet is also allowed, for example 2.5 feet.

Feet and inches entered

Total inches = (feet × 12) + inches. Then the app uses exact match or linear interpolation.

Interpolation works only inside the chart range. Outside the minimum or maximum chart values, the app shows an error.
